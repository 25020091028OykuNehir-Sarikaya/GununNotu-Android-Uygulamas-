# GununNotu-Android-Uygulamas-
Android Studio  ortamında Java dili  ile i7-13620H, 32GB RAM, RTX 5070 sistemde geliştirildi. EditText verisi Intent.putExtra() ile paketlenip DetailActivity'ye taşınır. İkinci ekranda getIntent() ile yakalanan not TextView'a set edilir; finish() mekanizmasıyla bellek dostu (Back Stack) geri dönüş sağlanır.
